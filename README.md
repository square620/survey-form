# Survey Form

A simple responsive survey form built while learning web development. This project helps me practice HTML structure, CSS styling, and form handling.

## The Idea behind this project 💡

This is my first project for the **freeCodeCamp Responsive Web Design Certification**. 

I wanted to build something clean and functional that works on both mobile and desktop. I chose a simple color theme (blue/white) to keep it professional and easy to read.

## What I learned 🧠

- How to structure a form with semantic HTML5 tags
- Working with CSS Flexbox for layout
- Making designs responsive (mobile-first approach)
- Basic accessibility (labels, alt text, ARIA)
- Using Git & GitHub to track my progress
